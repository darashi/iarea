= Iarea

A library for DoCoMo Open Iarea.

= Installation

  % gem install iarea

or add

  gem 'iarea'

on your Gemfile.

= Example of use

  require 'iarea'

  area = Iarea::Area.find_by_lat_lng(43.0568397222222,141.3478975)
  area.name # => "すすきの"
  area.areacode # => "00208"
  area.zone.name # => "北海道"
  area.prefecture.name # => "北海道"
  area.neighbors.map(&:name) #=> ["大通公園周辺", "大通東", "山鼻/藻岩周辺", "中島公園周辺"]
